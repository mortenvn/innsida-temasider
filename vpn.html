Koble til VPN ()

Endre operativsystem

Noen NTNU-ressurser krever at du er koblet til NTNUs nettverk for å bruke dem. Eksempler på dette er Bibsys, flere filservere og hjemmeområdet ditt. For å benytte disse ressursene hjemmefra er du nødt til å bruke en VPN-klient. VPN-klienten vil kryptere all datatrafikken din og sende den via NTNUs nettverk. På den måten kan du surfe som om du befinner deg på campus.

Instruksjoner

  1. Last ned VPN-klienten
  2. Kjør filen du lastet ned. Dette vil starte en installasjon av VPN-klienten (Cisco AnyConnect). Følg instruksjonene til programmet er ferdig installert.
  3. Start programmet Cisco AnyConnect Secure Mobility Client.
  4. Fyll inn sslvpn.ntnu.no i tekstfeltet som dukker opp og trykk Connect.
  5. Fyll inn ditt NTNU-brukernavn og passord. Trykk OK.
  6. Du er nå koblet til NTNUs nettverk. For å koble fra, åpner du Cisco AnyConnect igjen og trykker på Disconnect.

  1. Last ned VPN-klienten
  2. Legg filen du lastet ned i Hjem-mappen din. Hjem er en samlemappe som inneholder andre mapper som «Skrivebord», «Nedlastinger» og «Dokumenter».
  3. Åpne programmet Terminal. Kopier inn sudo ~/vpnsetup.sh og trykk enter. Dette vil starte installasjonen av VPN-klienten (Cisco AnyConnect).
  4. Tast inn ditt administratorpassord. Vanligvis er dette passordet du logger deg på PC-en din med. Etter dette kan du lukke vinduet.
  5. Start programmet Cisco Anyconnect Secure Mobility Client.
  6. Fyll inn sslvpn.ntnu.no i tekstfeltet som dukker opp og trykk Connect.
  7. Fyll inn ditt NTNU-brukernavn og passord. Trykk Connect.
  8. Du er nå koblet til NTNUs nettverk. For å koble fra, åpner du Cisco AnyConnect igjen, trykker på Connection-fanen og trykker på Disconnect.
Bor du på studentby? Det er ikke nødvendig å bruke VPN på studentbyene, siden disse nettverkene regnes som en del av NTNU-nettverket. Unntaket er Lerkendal studentby, hvor du fortsatt må bruke VPN for å koble til NTNU-nettet.
  1. Installer AnyConnect ICS+ fra Google Play
  2. Start appen AnyConnect ICS+.
  3. Trykk på Connection og deretter Add New VPN Connection.
  4. Velg en passende tittel for Description, for eksempel NTNU. For Server Address, tast inn sslvpn.ntnu.no. Til slutt, trykk på Done.
  5. Trykk på NTNU og skru deretter på AnyConnect VPN.
  6. Fyll inn ditt NTNU-brukernavn og passord. Trykk Connect.
  7. Du er nå koblet til NTNUs nettverk. For å koble fra, åpner du AnyConnect ICS+ igjen og skyver tilbake skyveknappen øverst på siden.
  1. Gå til App Store og installer Cisco AnyConnect
  2. Start appen AnyConnect.
  3. Trykk på Connections og deretter Add VPN Connection.
  4. Velg en passende tittel for Description, for eksempel NTNU. For Server Address, tast inn sslvpn.ntnu.no. Til slutt, trykk på Save.
  5. Trykk på Home og skru deretter på AnyConnect VPN.
  6. Fyll inn ditt NTNU-brukernavn og passord. Trykk Connect.
  7. Du er nå koblet til NTNUs nettverk. For å koble fra, åpner du AnyConnect igjen og skyver tilbake skyveknappen øverst på siden.

NTNU har dessverre ikke VPN-støtte for operativsystemet ditt. For å se på oppsettet til operativsystemene vi støtter, kan du trykke på Endre operativsystem ovenfor og velge et annet OS.
